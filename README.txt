Steps to install the project:

1. Clone or download the project in local storage as a zip file.
2. Extract the zip file.
3. Open Command Prompt in windows
4. Change directory to the project directory
5. type following commands to run the project:
	pip install requirements.txt for python version 2 or
	pip3 install requirements.txt for python version 3
6. Run the blockchain.py file by using command:
	python blockchain.py for python version 2 or 
	python3 blockchain.py for python version 3
7. If prompt to install the additional requirments intall it using pip command and again run blockchain.py file as mentioned above
8. open any browser and type the local ip addrress as shown in Command prompt
